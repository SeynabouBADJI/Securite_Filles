
# Module Intelligence Artificielle

Ce module est responsable de l'analyse des données, de l'entraînement des modèles
et de la prédiction du niveau de danger des lieux ainsi que des recommandations.

## Fonctions principales
- Prétraitement des données
- Prédiction du niveau de danger
- Recommandation de lieux sûrs
- Communication avec le backend

## Technologies
- Python
- Scikit-learn
- Pandas
